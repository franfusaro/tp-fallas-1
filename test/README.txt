1) Posicionarse en la carpeta y definir las siguientes variables:

$ export FLASK_APP=main.py
$ export FLASK_ENV=development

2) Iniciar la app: 

$ flask run

3) Abrir en el browser http://localhost:5000. Por ahora el botón para redirigir al diagnóstico no funciona, así que entrar a /consulta para ver el formulario
